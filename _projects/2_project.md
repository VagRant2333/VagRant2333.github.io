This is an on-campus innovation and entrepreneurship project supervised by Prof. Weifeng Ge.

The goal is to build an embodied assistant platform with a wheeled base and single-arm manipulation capability, then connect it with language-level tool-calling workflows.

Key contributions:

- Implemented a preliminary assistant pipeline for task decomposition and execution.
- Integrated robotic action interfaces with high-level tool-calling logic.
- Explored the bridge between language-driven planning and embodied manipulation.

This project reflects my long-term interest in reliable embodied agents that can operate in real-world assistance scenarios.
